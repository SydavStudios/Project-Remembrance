Project: Remembrance is the first game developed by Sydav Studios

The game is currently in Pre-Production so it is not likely to come out until at least late 2027

The game is a roguelike where you collect randomized items on your runs to upgrade yourself, unlock new abilities, or changes to your attacks

As you progress, you unlock more and more of the game and areas to explore.

There are new game plus' where the story is continued

The game is being developed by:

Sydney Davidson - game director, coder, artist

Alex W - Musician, head of audio design, artist

RadTaiga - artist, character designer
